# Group Study Adults Experiment 1

This repository contains the jsPsych web version of Experiment 1.

To run a specific counterbalance version, add `?version=1` through `?version=4`
to the page URL. Participant IDs can be passed with `?participant=...` or the
standard Prolific URL parameters.

Data are saved through DataPipe using experiment ID `7NbjHHkuurpH`.
